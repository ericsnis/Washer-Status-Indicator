# Boards
 The schematic folder contains the board and schematic files in Eagle format

## Files for surface mount (SMT) boards
- _Washer Machine Status Inland ESP32 1206.brd_ - Surface mount board file
- _Washer Machine Status Inland ESP32 1206.sch_ - Surface mount schematic file

## Files for through hole component boards
- _Washer Machine Status Inland ESP32 Through Hole.brd_ - Through hole component board file
- _Washer Machine Status Inland ESP32 Through Hole.sch_ - Through hole component schematic file

**You need _both_ the .brd and .sch files to open the board in Eagle.**
